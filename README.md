# Saikiran Rayudu — Portfolio 2026

Production-grade cinematic developer portfolio: React + Vite + TypeScript + Tailwind + Framer Motion.

## Run locally
```bash
npm install
npm run dev        # http://localhost:5173
npm run build      # production build
```

## Personalise
Edit `src/data/resume.ts` — one file controls everything (name, phone, email, projects, skills).

## Deploy to Vercel
```bash
npm i -g vercel && vercel
```
Or import GitHub repo at vercel.com → auto-detects Vite → Deploy.

## Features
- Dark mode default, premium soft light mode
- Smart chat widget: WhatsApp-style bot flows + free text + typing dots
- Mobile/desktop WhatsApp auto-detection
- 3D tilt project cards + glow hover
- Floating animated skill cards with shimmer
- Animated timeline for experience
- Cursor glow + scroll progress bar
- Section spy navbar
- Framer Motion stagger animations throughout
